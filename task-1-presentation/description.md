# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
**Canva** (canva.com) — a free, browser-based graphic design platform.

## What the Infographic Shows

The infographic is titled **"What is Digital Literacy?"** and is divided into three sections:

1. **What Digital Literacy Means** — Defined as the ability to find, evaluate, use, share, and create content using digital technologies and the internet responsibly and safely.

2. **Useful Digital Tools for Students** — Highlights platforms like Google Workspace (Docs, Sheets, Forms), GitHub for code management, Canva for design, and HackerRank for coding practice.

3. **Safe Internet Practices** — Covers the importance of strong passwords, enabling two-factor authentication, recognising phishing emails, and being careful about what personal information is shared online.

The design uses a blue-and-white colour scheme with simple icons to make information visually accessible.

## Reflection

One thing I found both interesting and challenging was understanding **visual hierarchy** — deciding what the viewer should notice first, second, and third. Arranging the layout so the most important information stands out without making the design look cluttered required several rounds of trial and error. I experimented with different colour combinations and font sizes before arriving at the final design. This task taught me that good communication is not just about what you say, but how you present it.
